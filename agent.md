# Agent Operating Notes

Use this file as the short agent entrypoint.

## Product Principles

- Citizen-first: plain language, large touch targets, mobile-first.
- Source-first: every answer carries provenance and freshness.
- Conservative: show uncertainty and missing fields.
- Legal: official APIs/downloads first, controlled scraping last.
- Accessible: keyboard navigation, readable contrast, low-bandwidth patterns.
